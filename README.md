# 👨‍💻 Sobre mí

Soy **Ingeniero en Sistemas y desarrollador Full Stack** con experiencia construyendo aplicaciones web completas, sistemas empresariales y plataformas internas.

Mi enfoque principal es desarrollar **software escalable, mantenible y bien estructurado**, priorizando:

- Arquitectura sólida
- Rendimiento
- Código limpio
- Experiencia del usuario
